
An Android application that enables users to complete tasks, earn rewards, and request withdrawals through UPI.

-> Key Features

* User Authentication ( Using Firebase Auth )
* Task Management System 
* Wallet & Earnings Tracking
* UPI Withdrawal Requests
* Firebase Integration
* Duplicate Account Detection ( Two Different Device, Same Account )
* Auto Task Detection
* Real-time Data Updates ( Using Firebase Firestore )

-> Tech Stack

* Java
* Android SDK
* Firebase Authentication
* Firebase Firestore
* Gradle

-> What I Learned

* Android App Development
* Firebase Backend Integration
* Authentication & Database Management
* UI/UX Design
* Real-time Data Handling
* How Real Project Works.
  

This project demonstrates my ability to build a complete Android application with user management, task tracking, and payment-related workflows.
